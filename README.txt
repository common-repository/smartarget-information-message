== Description ==

Information Message - The right way to communicate information with users
Smartarget developed a plugin for Wordpress so you can communicate information to users in a visible way, yet not intrusive.
The Information Message will be shown to users on all pages, both on desktop and mobile, and will allow you to communicate with them relevant information that you believe is important for them to know.
Smartarget Information Message  plugin works also with WooCommerce, Elementor and Oxygen Builder.

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
